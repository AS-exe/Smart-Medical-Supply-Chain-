# Smart Medical Supply Chain (SMSC) Simulator 🏥🚨

An AI-driven simulation platform designed to forecast medical supply demand during disaster scenarios and optimize emergency distribution planning. 

## Overview
During emergencies, the rapid and accurate allocation of medical resources is critical. This project utilizes machine learning to predict supply needs based on disaster data. It features a robust predictive backend powered by XGBoost and an interactive, real-time frontend built with Streamlit.

## Key Features
* **Predictive Modeling:** Utilizes **XGBoost** to forecast medical supply demands with high accuracy based on historical disaster data and simulated inputs.
* **Data Preprocessing Pipeline:** Includes structured feature engineering and data cleaning using **Pandas** and **NumPy** to ensure model reliability.
* **Interactive Dashboard:** A dynamic **Streamlit** web application that allows users to input disaster parameters and visualize demand forecasts in real-time.

## Technology Stack
* **Language:** Python
* **Machine Learning:** XGBoost, Scikit-learn
* **Data Manipulation:** Pandas, NumPy
* **Frontend / Visualization:** Streamlit
